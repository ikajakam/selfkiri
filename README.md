### Prerequisites

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/MEY-D/selfkiri.git
   cd selfkiri
   ```
2. **Install dependencies:**
   ```bash
   npm install
   npm install express
   ```
3. **Run the app:**
   ```bash
   node app.js
   ```
4. **Open in your browser:**
   [http://localhost:3000](http://localhost:3000)
